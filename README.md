# 📌 CRUD-Senai

Um projeto de **CRUD (Create, Read, Update, Delete)** desenvolvido durante as aulas do **SENAI**, composto por:

✔️ Backend (API)  
✔️ Frontend (interface web)  

Este projeto serve como exemplo de aplicação full-stack que permite gerenciar dados de forma simples utilizando JavaScript, HTML e CSS no front e uma API no back.

---

## 🧠 Sobre o projeto

Este repositório contém duas partes principais:

### 1. 🖥️ Backend – API
Localizado em `/crud-senai-api`, responsável por fornecer os endpoints da aplicação.

### 2. 🌐 Frontend – Aplicação Web
Localizado em `/crud-senai-front`, interface que consome a API para realizar operações CRUD.

---

## 🚀 Tecnologias usadas

Frontend:
- ⭐ HTML
- 🎨 CSS
- 💻 JavaScript

Backend:
- 🛠️ Node.js (presumido pelo `package.json`)
- 📦 Dependências gerenciadas com npm
- 🗃️ API REST para operações CRUD

---

## 🧾 Funcionalidades

✔️ Criar novos registros  
✔️ Listar registros existentes  
✔️ Atualizar registros  
✔️ Deletar registros  

Essas quatro operações formam o ciclo básico de um CRUD e permitem a manipulação completa dos dados.

---

## 🛠️ Como usar (desenvolvimento)

### 🔹 Backend

1. Abra o terminal na pasta `crud-senai-api`  
2. Instale as dependências:
```bash
npm install
````

3. Configure variáveis de ambiente (ex.: `.env`) se necessário
4. Inicie a API:

```bash
npm start
```

---

### 🔹 Frontend

1. Abra o terminal na pasta `crud-senai-front`
2. Abra o arquivo `index.html` no navegador ou use um servidor local
3. A aplicação deve se conectar à API e permitir operações CRUD
